# UART-Controller-SystemVerilog
Designed and verified a UART Controller using SystemVerilog implementing asynchronous serial communication with FSM-based TX/RX architecture and simulated in Xilinx Vivado.

# Overview
The UART Controller project is designed and implemented using SystemVerilog to enable asynchronous serial communication between digital systems. The design includes separate UART Transmitter (TX) and UART Receiver (RX) modules controlled using Finite State Machine (FSM) architecture.

The transmitter converts parallel input data into serial output, while the receiver converts serial data back into parallel form. The functionality is verified through a testbench simulation in Xilinx Vivado, ensuring accurate data transmission and reception.

This project demonstrates fundamental RTL design concepts, digital communication protocols, state machine implementation, and verification techniques, making it a strong beginner-to-intermediate VLSI portfolio project.

# Tool Used
* Language     : SystemVerilog
* IDE          : Xilinx Vivado
* Simulator    : XSim
* Design Style : RTL + FSM
* Version Ctrl : Git / GitHub
